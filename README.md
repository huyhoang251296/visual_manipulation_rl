# To use Moveit Setup Assisstant

1. Setup XDG_RUNTIME_DIR
```bash
export XDG_RUNTIME_DIR=/your/chosen/directory

# Change mode of the folder to 700
chmod -R 700 /your/chosen/directory
```

2. Install ubuntu-desktop

```bash
sudo apt-get install ubuntu-desktop
```

3. Try to load the URDF many times
